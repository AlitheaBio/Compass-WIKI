# HLA-Compass Public Documentation

This directory contains the source code for the **external** HLA-Compass Module Developer SDK documentation, hosted at [docs.alithea.bio](https://docs.alithea.bio).

## Structure

*   `index.md`: Landing page.
*   `guides/`: User guides for module developers (Getting Started, Auth, Testing).
*   `reference/`: API reference documentation for the `hla_compass` Python package.

## Internal Documentation

Documentation for **platform developers**, architecture, infrastructure, and internal operations has been moved to the [`wiki/`](../wiki/) directory.
